# 学生管理系统

## 1.项目需求

### 1.1 功能

1. 新增学生记录
2. 列出所有学生
3. 按班级查询学生
4. 按姓名查询学生
5. 按学号查询学生
6. 按姓名删除学生
7. 按学号删除学生
8. 修改学生班级
9. 修改学生卡号
0. 退出系统


### 1.2 技术栈

编程语言：Python语言
数据存储：Mysql数据库
数据操作：SQL语言
交互界面：基于现学的命令行模式
