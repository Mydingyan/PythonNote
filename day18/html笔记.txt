
# 超文本标记语言
超文本： 不止于文本，还有与音频、图片、视频


标记 ：<h1> 标题 </h1>
<!DOCTYPE html>  ：表示网站说明


HTML两大部分
head
body

主要有标签（页面元素）组成
head标签
1.title

body标签中的内容
1.h1标签  -- 标题
2. p标签 --  行标签
3. img标签  -- 图片
4. table标签 -- 表格
5. a标签 -- 超链接
6. input -- 输入框
7. select  -- 下拉框
8. div  --  （说白了就是盒子，层层之间能嵌套，方便排版）
9. iframe -- 子窗口
